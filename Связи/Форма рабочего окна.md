__MainWorkWindow__ — рабочее окно:

- Слева: дерево навигации по модулям (Финансы, ОС, Материалы и т.д.)
- Центр: `ContentControl`, который переключает UserControl'ы
- Из рабочего окна доступны диалоги профиля и управления пользователями

__Левая панель (Header + TreeView)__

- Шапка: логотип системы, название, кнопка смены темы, логотип и название текущей инфобазы
- Дерево модулей: строится по `MetadataModules.IsActive` для текущей инфобазы
- Каждый модуль содержит дочерние элементы: справочники, документы, отчеты
